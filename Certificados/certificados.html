Complete Python Bootcamp - Go From Zero to Hero in Python 3

Com aproximadamente 24h de curso, este abrange desde a instalação da linguagem Python na máquina até tópicos avançados na mesma. Utiliza o ambiente Jupyter Notebook para um aprendizado mais dinâmico de Python e contém vários projetos para o desenvolvimento das habilidades nas diferentes funcionalidades da linguagem. Contém também tópicos como envio de e-mail utilizando código, webscraping, navegação por pastas e introdução a GUI's.

Concluído em Jan/2020 (Clique aqui para visualizar o certificado)

The Complete SQL Bootcamp - Go From Zero to Hero

Com aproximadamente 9 horas totais de curso, este abrange o uso da linguagem de consulta em bancos de dados utilizando o ambiente PostgreSQL. Inclui desde a instalação das ferramentas necessárias até o ensino das funcionalidades da linguagem para obter-se os dados da maneira correta. Também abrange a criação de Bancos de Dados e a utilização de Python em conjunto com o ambiente PostgreSQL.

Concluído em Out/2020 (Clique aqui para visualizar o certificado)

Composição de Programas em Python

Curso fornecido durante a pandemia pelo Departamento de Ciências da Computação da UFMG, que abrange 3 principais tópicos da linguagem Python: Orientação a Objetos, Programação Funcional e Multiprocessamento. Cada um desses tópicos é lecionado por um professor diferente e é dividido em subtópicos em adição a exercícios para a aplicação do que foi aprendido.

Concluído em Jul/2020 (Clique aqui para visualizar o certificado)

Microsoft Excel - Excel From Beginner to Advanced

Com aproximadamente 18h de curso, este abrange todos os tópicos na utilização da ferramenta Excel, da Microsoft. Ensina desde a organização da área de trabalho do software até tópicos mais avançados como a confecção de Macros e a utilização do VBA. Divide-se em 3 níveis: Iniciante, Intermediário e Avançado para que o aprendizado evolua da mais simples à mais complexa ferramenta disponível no software.
